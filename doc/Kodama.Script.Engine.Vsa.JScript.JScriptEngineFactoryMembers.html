JScriptEngineFactory 概要

パブリック コンストラクタ

JScriptEngineFactory コンストラクタ JScriptEngineFactory クラスの新しいインスタンスを初期化します。

パブリック メソッド

CanCompile 指定されたファイルがコンパイル可能かチェックする
Create スクリプトエンジンのインスタンスを生成する
CreateFromFile ファイルからスクリプトエンジンのインスタンスを生成する
