IScriptEngine メソッド

IScriptEngine インターフェイスのメソッドは以下のとおりです。IScriptEngine インターフェイスの完全なメンバ一覧については、 IScriptEngine メンバ のトピックを参照してください。

パブリック メソッド

AddGlobalIntstance スクリプトからみえるグローバルなインスタンスを追加する
AddReference スクリプトで参照するアセンブリを追加する
AddScriptCodeFromFile スクリプトをファイルから読み出し、既にあるコードに追加する
AddScriptCodeFromText テキストを既にあるコードに追加する
Close スクリプトのリソースを開放し、スクリプトエンジンを無効にする
Compile スクリプトをコンパイルする
Run スクリプトを実行する
Stop スクリプトの実行を停止する
