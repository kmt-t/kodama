CSharpScriptEngine.Run メソッド 

[This is preliminary documentation and subject to change.]

スクリプトを実行する

public void Run();

実装

IScriptEngine.Run

解説

スクリプトを実行するにはエントリーポイントが必要です。 エントリーポイントとはスクリプトの一番最初に起動する メソッドのことをいいます。このエンジンでのC#Scriptの エントリーポイントは、関数名が"Main"で引数なしの静的な メソッドで、ScriptEntryPointAttribute属性がついているものとします。

例外

例外の種類 条件
ScriptEntryPointNotFoundException エントリーポイントがみつからない場合はこの例外を投げる
NotCompiledException スクリプトがコンパイルされていない場合はこの例外を投げる
