CompileErrorInfo プロパティ

CompileErrorInfo クラスのプロパティは以下のとおりです。CompileErrorInfo クラスの完全なメンバ一覧については、 CompileErrorInfo メンバ のトピックを参照してください。

パブリック プロパティ

Description エラーの説明文
ErrorLine エラーの発生した行
ErrorText エラーの発生した行のテキスト
SourceName エラーの発生したソース名
